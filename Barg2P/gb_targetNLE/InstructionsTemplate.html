In this part, you must indicate the position of a number on a line. You will do this exercise {{ C.NUM_ROUNDS }} times, with a different number each time. The line goes from 0 to 100, and the cursor is initially positioned at 0. You will have {{ C.DECISION_TIME }} seconds to move the cursor, using the mouse, to get closer to the target value. If you have not responded by the end of the {{ C.DECISION_TIME }} seconds, the program will consider your answer to be 0 and will automatically move on to the next exercise.

Calculation of gain

The gain depends on the distance between the chosen position and the target number. Precisely, if d is the distance between the chosen position and the target number, the gain is equal to {{ subsession.constante|cu }} - {{ C.FACTEUR_DISTANCE }} x d, with d = |target number - chosen position|. Let's illustrate this calculation with three examples.

Example 1: the target number is 12.2, and you position the cursor at the value 11.5.
Your gain is then equal to {{ subsession.constante|cu }} - {{ C.FACTEUR_DISTANCE }} x |12.2 - 11.5| = €2.96.

Example 2: the target number is 12.2, and you position the cursor at the value 55.
Your gain is then equal to {{ subsession.constante|cu }} - {{ C.FACTEUR_DISTANCE }} x |12.2 - 55| = €0.86.

Example 3: the target number is 12.2, and you position the cursor at the value 90.
Your gain is then equal to {{ subsession.constante|cu }} - {{ C.FACTEUR_DISTANCE }} x |12.2 - 90| = -€0.89. In this case, your gain is €0.

At the end of this part, one of the {{ C.NUM_ROUNDS }} exercises will be randomly selected by the computer program. Your gain in this exercise will determine your gain for this part. The minimum gain is {{ 0|cu }} and the maximum gain is {{ subsession.constante|cu }}.
