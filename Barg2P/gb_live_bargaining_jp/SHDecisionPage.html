  • 両方のプレイヤーが選択1をプレイする場合、それぞれ {{ both_stag_payoff }} ECUを獲得します。
  • プレイヤー1が選択1を、プレイヤー2が選択2をプレイする場合、プレイヤー1は {{ stag_hare_payoff }} ECUを、 プレイヤー2は {{ hare_stag_payoff_p2 }} ECUを獲得します。

  • 両方のプレイヤーが選択2をプレイする場合、プレイヤー1は {{ both_hare_payoff_p1 }} ECUを、プレイヤー2は {{ both_hare_payoff_p2 }} ECUを獲得します。
